-------------------------------------------------------------------------------------------------------------
How To install:
open .env file - set DataBase/Busher/Mail ... data
run migration in command window/terminal (php artisan migrate)
run seeds command window/terminal (php artisan db:seed)
run server command window/terminal(php artisan serv)
open browser and get link 127.0.0.1:8000/myadmin
set login data (E-mail/Password/Secret Key)
after that you must login -> get admin using link 127.0.0.1:8000/myadmin/{secret key}
login with your data and create your amazing system.
